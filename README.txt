Integration Contact Form 7 and CRM Bitrix24 plugin.

== Description ==

Integration Contact Form 7 and CRM Bitrix24 plugin. Very flexible and easy to configure. You can create an unlimited number of forms and transfer their values to Birtix24

Brief Instruction:

1. Install the "Contact Form 7" plugin and "CF7 to Bitrix24 integration"
2. Go to Contact Form7 -> Bitrix24 integration
3. Enter your details for connecting to Bitrix24, follow the prompts:
	3.1 Bitrix Domain - NAME.bitrix24.ru
	3.2 Port - Typically 443
	3.3 REST api path - /crm/configs/import/lead.php
	3.4 Login - Login to enter Bitrix24
	3.5 Password - Password to enter Bitrix24
	3.6 Save Settings
4. Go to one of your created forms in "Contact Form 7" and go to the Bitrix24 leads tab
5. Specify which fields to send the form values to. Save Settings

Terms of Use:

This plugin provides basic functions. We have a paid add-on - <a href="https://kowalski.su/product/cf7-to-bitrix24-integration/" target="_blank">https://kowalski.su/product/cf7-to-bitrix24-integration/</a>. It extends the functionality of the main plugin:

2. Ability to configure UTM tags for collecting statistics in Yandex metrics.


== Installation ==

1. Unzip the contents of the zip file to the plugins folder of your site (wp-content/plugins/) using your favorite FTP program.
2. Activate the plugin on the "Plugins" page in the admin panel.
3. Everything, installation is completed.
